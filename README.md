# Landing Page — 신사임당 포트폴리오

다크 테마의 현대적인 포트폴리오 / 랜딩 페이지입니다.

## 개요

"신사임당" 테마의 개인 포트폴리오 페이지로, 부드러운 스크롤 효과와 CSS 커스텀 변수를 활용한 세련된 다크 모드 디자인을 적용했습니다.

## 주요 특징

- 다크 테마 디자인 (CSS 커스텀 프로퍼티 활용)
- 부드러운 스크롤 애니메이션
- 반응형 레이아웃
- 순수 HTML/CSS — 별도 빌드 도구 불필요

## 기술 스택

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)

## 실행 방법

`index.html` 파일을 브라우저에서 직접 열거나, 로컬 서버를 통해 실행하세요.

```bash
# Python 내장 서버 사용 예시
python -m http.server 3000
```
